# Terry's Trousers Profit Maximizer

An interactive business analytics dashboard built with HTML, Tailwind CSS, JavaScript, and Chart.js.
Live Demo https://ayeshakhawar217-cmd.github.io/Ecommerce-profit-maximizer/
## Features

- Real-time profit modeling
- Interactive scenario analysis
- Return-rate optimization
- Revenue and cost projections
- Dynamic chart visualization
- Scenario saving using Local Storage
- CSV export functionality

## Technologies

- HTML5
- Tailwind CSS
- JavaScript
- Chart.js
- Local Storage API

## Business Use Case

This dashboard helps decision-makers evaluate how return rates, order volumes, return fees, and marketing spend impact profitability.
